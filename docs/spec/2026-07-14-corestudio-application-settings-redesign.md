# CoreStudio 应用设置结构

> 状态：当前有效

应用设置按用户目标组织，不展示底层协议矩阵。

## 页面

1. 图像生成：当前服务、模型、API Key 和服务详情。
2. Codex 集成：安装、更新、修复、版本检测和打开当前项目。
3. 关于：客户端版本、仓库地址、开源依赖和版本。

## 规则

- 当前服务选择立即生效。
- API Key、模型和 API 地址修改需要显式保存。
- 离开有未保存修改的页面时要求确认。
- Codex 集成只执行应用包内固定安装器，不接受 renderer 提交任意命令。
- Bridge 地址、project token 等诊断信息不进入默认页面。
- 画布中不放置重复的 Codex 状态或设置入口。

## 验收

- 普通用户无需理解内部协议即可完成图像服务和 Codex 集成配置。
- 设置中不存在外部 Agent 运行时、会话或调试配置。
- 关于页的信息来自打包元数据，版本与实际应用一致。
