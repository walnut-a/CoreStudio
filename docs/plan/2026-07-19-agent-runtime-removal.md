# CoreStudio 内置 Agent runtime 移除记录

> 状态：已完成

## 产品边界

CoreStudio 本体只保留单次图片生成。需要分析、连续迭代、并行生成或组合工具的任务统一从 Codex 发起，通过 CLI / Local Bridge 读取和写回 CoreStudio 项目。

## 完成内容

- 删除内置 Agent 进程、会话、任务、日志、设置、IPC 和调试能力。
- 删除 renderer 中的模式切换、Agent 对话、运行状态、上下文组件和专属样式。
- 本地生成器收敛为 builtin-only，保留引用确认、不可空提交、提交后清空和并发生成。
- 删除退役协议的历史来源类型、读取兼容、展示文案、迁移代码和测试数据。
- 新生成记录只允许 `corestudio` 和 `agent-board` 两种来源。
- 项目图片资产和画布引用不在清理范围内。

## 验证标准

- TypeScript 类型检查已通过。
- 桌面端 195 个测试文件、1545 项测试已全部通过，无跳过项。
- Renderer 与 Electron 生产构建已通过。
- 活动代码、测试和当前文档不再包含退役协议的类型、路由或产品入口。
