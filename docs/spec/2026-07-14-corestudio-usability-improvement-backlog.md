# CoreStudio 易用性优化清单

> 状态：持续维护

## 已完成

- 设置按图像生成、Codex 集成和关于三个用户目标组织。
- Codex 集成提供固定安装器、真实检测和打开当前项目入口。
- Agent Board 移除重复生成输入器，任务回到 Codex 对话发起。
- 本地输入框恢复单次生成语义：候选提示词不可直接提交、提交后清空、多任务可并行。
- 左侧统一显示普通生成、Codex 写回和旧 Agent 图片记录。
- 项目写回增加事务、旧快照冲突保护和恢复检查。

## 后续观察

- 图像服务首次配置是否需要轻量连通性检查。
- 生成失败文案是否能稳定区分配置、网络、额度和服务端错误。
- Codex 集成版本变化时，更新提示是否足够明确。
- 大项目的图片加载、缩略图缓存和项目修复性能。

## 不做

- 不把 CoreStudio 变成多 Agent 调度平台。
- 不允许 Agent 直接读取 API Key 或修改项目文件。
- 不在 Agent Board 中复制桌面端生成和设置入口。
